# Simple Portfolio

This is a simple website portfolio built only using React.js and TailwindCSS as the tech stacks.

## Features
- Responsive design
- Fast build with Vite
- Clean and modular components

## Installation
Clone the project

```bash
git clone https://github.com/Dhorq/simple-portfolio.git
```

Navigate to the project directory:
```bash
cd simple-portfolio
```

Install dependencies:
```bash
npm install
```

Run the project locally:
```bash
npm run dev
```
